**What can a scheduled learning event be?**

Anki already solved *when* a memory should return. Ritornello works on what
returns: cards that grade a drawn coastline or a transcribed melody, agents
that author and repair curricula against a live collection, and the toolchain
that makes ambitious cards safe to ship on desktop, iOS, and Android.

- **Decks** — interactive geography, sight-singing and dictation,
  quantitative drills, and other decks built as small applications.
- **Add-ons** — fractional new-card scheduling, workload triage, an
  in-review AI chat dock, and safe collection operations.
- **Toolchain** — disposable-profile testing across all three Anki
  platforms, APKG-vs-live diffing, and review-first release automation.

Everything is dogfooded daily against a collection of ~257k notes before it
ships. Start with the manifesto: *The Shape of Anki to Come* (link when
published), or browse the pinned repositories below.
